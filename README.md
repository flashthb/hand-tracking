# Count Fingers - Hand Tracking Project

Here is the raw code for the hand tracking and finger counter. Everything is inside `main.py` and the `modules/` folder in case you want to copy the files.

I want to improve it in the future to control your PC with your hands.

## Requeriments
* **Python 3.8** or newer
* **Webcam** integrated or USB
* **Libraries** in `requeriments.txt`

**Careful:**
We are going to use MediaPipe 0.10.10, so probably your will have to install a lower version of python (3.10.11 in my case).

## Build Instructions:
1. **Clone the repository:**
    ```bash
    git clone https://github.com/flashthb/hand-tracking.git
    ```

2. **CD into the newly cloned directory:**
    ```bash
    cd ./hand-tracking
    ```

3. **Install dependecies:**
    ```bash
    pip install -r requirements.txt
    ```

4. **Run the program:**
    ```bash
    python main.py
    ```
